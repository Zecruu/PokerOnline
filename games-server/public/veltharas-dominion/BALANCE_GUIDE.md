## Game Balance Settings

| Setting | Value | Description |
|---------|-------|-------------|
| Enemy Health Mult | 0.35 | Lower base health for easier early game |
| Enemy Damage Mult | 1.0 | Standard damage |
| Enemy Speed Mult | 1.3 | Faster enemies for exciting gameplay |
| Spawn Rate Mult | 0.5 | Faster spawns (lower = more frequent) |
| Scaling Per Wave (Early) | 12% | Per wave scaling for waves 1-9 |
| Scaling Per Wave (Late) | 90% | Per wave scaling for waves 10+ |
| Late Game Wave | 10 | When late game scaling kicks in |
| Player Health Mult | 1.0 | Standard player health |
| XP Mult | 1.2 | More XP for faster leveling |
| Player Speed Mult | 1.15 | Faster player movement |

### Post Wave 15 Scaling
- Enemies are 50% bigger
- Enemies have 50% more HP and damage
