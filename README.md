## How to Play

1. **Create a Room**
   - Click "Create New Room"
   - Enter your name
   - Share the 6-digit room code with friends

2. **Join a Room**
   - Click "Join Room"
   - Enter the room code and your name
   - Wait for the host to start the game

3. **Play Poker**
   - Each player starts with $1000 in chips
   - Standard Texas Hold'em rules apply
   - Use Fold, Check, Call, or Raise actions
   - Win the pot with the best hand!

## Running Locally

Simply open `index.html` in your web browser. No build process or server required!

```bash
# Or use a local server for better experience
python -m http.server 8000
# Then visit http://localhost:8000
```

## Game Rules

- **Blinds**: Small blind $10, Big blind $20
- **Betting Rounds**: Pre-flop, Flop, Turn, River
- **Hand Rankings**: Royal Flush, Straight Flush, Four of a Kind, Full House, Flush, Straight, Three of a Kind, Two Pair, One Pair, High Card
